Welcome to Kerkers en Draken! > Help
  Available Actions:
  Look : Shows the location's description with visible objects, all possible exits, and any enemies present.
  Search* : Moves hidden objects in the current location to visible objects, except those associated with enemies here.
  Go <north|south|east|west>* : If there's an exit in the chosen direction, move there. If enemies are present, they each attack once before moving.
  Take <OBJECT> : Moves a visible object from the location to the player. For 'gold coins', add to the playerâ€™s total and delete the object.
  Drop <OBJECT> : Moves an object from the player to visible objects in the current location.
  Inspect <OBJECT> : Shows description of the object if visible in the location or held by the player.
  Inspect <ENEMY> :
    If enemy has health: Shows the enemy's description (if present in the location).
    If enemy has 0 health: Shows the description, and moves hidden items from the enemy to visible objects in the location.
  Inspect Self : Displays player stats: health, attack chance, gold, equipped weapon, armor, and inventory.
  Attack <ENEMY>* : Attack the specified enemy with the equipped weapon.
  Equip <OBJECT:ARMOR>* : If player owns the armor, equip it. Previously worn armor becomes a visible object in the location.
  Equip <OBJECT:WEAPON> : If player owns the weapon, equip it. Previously held weapon becomes a visible object in the location.
  Wait* : Skip the turn. Enemies may attack and move.
  Consume <OBJECT:CONSUMABLE> : Effects vary based on potion type:
    Health Elixir: Restores 5-15 health points.
    Experience Potion: Increases attack chance by +10%, up to 90% max.
    Teleportation Potion: Teleports player to a random location.
  Help : Shows this list of available actions.
  Godmode : Toggles 'godmode' where player cannot lose health or miss attacks.
  Quit : Ends the game and closes the application properly to reveal any memory leaks.

* Indicates an action that ends the player's turn.
